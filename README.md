## Microservices Migration of Chat Backend — Spring Cloud
`Spring Boot` `Spring Cloud` `Eureka` `API Gateway` `Feign` `JWT` `MySQL`

- Decomposed monolithic chat backend into Auth, User, Message services + API Gateway
- Implemented service discovery (Eureka), centralized JWT validation at gateway, Feign inter-service calls
- Applied database-per-service pattern and client-side load balancing with Spring LoadBalancer
